This directory should contain the following icon files for the PWA:

- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)

## Additional Icons:
- symptom-icon.png (96x96 pixels) - For symptom checker shortcut
- fitness-icon.png (96x96 pixels) - For fitness shortcut
- clinic-icon.png (96x96 pixels) - For clinics shortcut
- badge-72x72.png (72x72 pixels) - For notification badge
- checkmark.png - For notification actions
- xmark.png - For notification actions

## Screenshots:
- screenshot1.png (1280x720 pixels) - Wide screenshot
- screenshot2.png (750x1334 pixels) - Narrow screenshot

## Design Guidelines:
- Use medical/healthcare theme with colors: blue (#3B82F6), green, white
- Include a heartbeat or medical cross symbol
- Ensure icons are recognizable at small sizes
- Use transparent background where appropriate
- Follow modern, clean design principles

## Note:
These icons need to be created manually using image editing software or generated from an online icon generator. The app will function without them, but the PWA installation experience will be incomplete.
